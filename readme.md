# 网上法律咨询系统

### 开发环境

- 开发语言： 后端 - Java(jdk1.8)，Python(3.7)，前端 - Html，JavaScript     
- 使用框架：Spring Boot(2.1.3)，MyBatis，Junit，pytorch(1.4)
- 数据库：mysql(8.0)
- 项目管理：Maven
- IDE：JetBrains Intelij IDEA 2020
- 开发系统：Windows10
- 网络服务器：Tomcat(9.0)
