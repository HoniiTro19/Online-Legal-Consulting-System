## 需求分析

### 展示部分

#### 首页
- 首页的轮播（轮播文章）点击之后跳转到普通文章页。【标题，内容摘要，图片】
- 最新文章： 按照发布时间来进行倒序排序展示（通过数据库 order by）
- 博客专栏： 展示前6个专栏。
- 推荐文章： 展示浏览量最大的文章。

#### 文章和专栏页
- 大家都能看，不需要登录，展示文章
- 专栏页，展示专栏信息
- 文章页评论功能 （要留联系方式，内容评论）
- 文章分页功能
- 文章页展示浏览量

#### 后台管理页面
- 登录功能（博主自己登录就行）
- 文章管理：
    - 专栏的信息添加和修改。
    - 文章发布
    - 文章删除
    - 文章修改

- 专栏管理：
    - 新增专栏
    - 修改专栏
    - 删除专栏：注意，删除之后，文章不能一起删除，删除对应的文章专栏信息即可。
